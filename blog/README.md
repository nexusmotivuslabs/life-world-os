## Adding New Posts

1. **Choose category**: Place in appropriate topic/genre folder
2. **Create hierarchy**: Use subdirectories for organization
3. **Naming**: Use kebab-case for filenames
4. **Metadata**: Include tags, date, context in frontmatter
5. **Update index**: Add to this README
