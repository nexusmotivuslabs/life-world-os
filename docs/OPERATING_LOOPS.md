# Operating Loops

## Weekly Loop

### Purpose
Regular check-in to maintain awareness and make small adjustments.

### Steps

1. **Review Current State** (5 minutes)
   - Check current season and days in season
   - Review cloud strengths
   - Check resource levels (especially Water)
   - Review category XP levels

2. **Record Activities** (10 minutes)
   - Log any activities from the week
   - Record XP gains
   - Update resources if needed

3. **Check Balance** (2 minutes)
   - Review balance indicator
   - Note any warnings
   - Consider recommendations

4. **Adjust if Needed** (3 minutes)
   - Update cloud strengths if significantly off
   - Plan activities for next week aligned with season

**Total Time:** ~20 minutes weekly

---

## Quarterly Loop

### Purpose
Strategic review and planning for the next quarter.

### Steps

1. **Review Season History** (15 minutes)
   - Review all seasons from past quarter
   - Note achievements and challenges
   - Identify patterns

2. **Check Milestone Progress** (10 minutes)
   - Review achieved milestones
   - Check progress toward next milestones
   - Plan milestone-focused activities

3. **Assess Engines** (15 minutes)
   - Review engine performance
   - Check fragility scores
   - Consider adding/removing engines
   - Plan engine improvements

4. **Plan Next Season** (20 minutes)
   - Decide which season to enter next
   - Plan season-specific activities
   - Set season goals

5. **Review Resources** (10 minutes)
   - Check Oxygen (months covered)
   - Review Gold accumulation
   - Assess Keys unlocked
   - Plan resource allocation

6. **Check Balance** (5 minutes)
   - Review category levels
   - Address any imbalances
   - Plan balanced activities

**Total Time:** ~75 minutes quarterly

---

## Monthly Check-In (Optional)

### Purpose
Mid-cycle check to ensure you're on track.

### Steps

1. Quick resource check
2. Review recent activities
3. Check if season transition needed soon
4. Adjust cloud strengths if needed

**Total Time:** ~10 minutes monthly

---

## Daily Micro-Loop (Optional)

### Purpose
Quick daily awareness without heavy tracking.

### Steps

1. Record any significant activity
2. Quick check of Water level
3. Note current season focus

**Total Time:** ~2 minutes daily
